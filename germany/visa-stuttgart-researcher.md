# Stuttgart 研究者向け Residence Permit 申請レポート

対象：非EU研究者（Visiting researcher ※博士課程学生扱いではない）
滞在都市：Stuttgart
所属例：University of Stuttgart

---

# 1. ドイツ到着後の手続きの全体像

ドイツでは以下の順番で手続きを行った。
1. 住民登録（Anmeldung）
2. Residence Permit申請 (オンライン)
3. 面談・書類提出
4. eAT発行

担当機関: Ausländerbehörde Stuttgart
(Eberhardstr. 39, 70173 Stuttgart)

# 4. Residence Permit 申請

## 事前予約

サイト: [https://www.stuttgart.de/buergerinnen-und-buerger/migranten/informationen-der-auslaenderbehoerde/kontakt-und-antrag/antrag-stellen#/#bueOverview](https://www.stuttgart.de/buergerinnen-und-buerger/migranten/informationen-der-auslaenderbehoerde/kontakt-und-antrag/antrag-stellen#/#bueOverview)

カテゴリー: 2. Aufenthaltstitel für die Erwerbstätigkeit (Residence permit for employment ※学生ではない扱いでvisaを申請するので)

Stuttgart市の公式オンライン予約サイトから要求される書類を提出する。提出すると、後日面談呼び出しのメールがくる。
予約の際に、
* パスポート
* 学位証明書など（必須ではなさそうだが提出した）
* invitation letter
* hosting agreement
* 収入証明（とりあえず事前に用意していた、滞在費は研究費から出るという内容の書類を出した）
* 健康保険証明書（この時点では必須ではないがあるに越したことはない）
をアップロードする必要がある。
ドイツ語能力証明書類の欄があるが必須ではないので無視して良い。

また、久慈は上記のサイトではなく、以下のサイトから必要そうだと思った書類を適当に添付して文章でvisa申請の予約を取りたいと問い合わせたら、2ヶ月後に面談の日程が来た。上記のサイトの方が公式の手順かと思われる。

サイト: [https://service.stuttgart.de/lhs-services/kontakt-auslaenderbehoerde/](https://service.stuttgart.de/lhs-services/kontakt-auslaenderbehoerde/)

## 面談

最初の予約申請から2ヶ月後に日時指定の面談日決定のメールが来る。ドイツ語話者を通訳として連れてくるように案内がある。久慈は研究室の友人達に助けてもらった。

### 面談の際に提出した書類

* パスポート
* 学位証明書など (担当者によっては必要ない場合もある)
* Invitation letter
* Hosting agreement
* 収入証明（とりあえず事前に用意していた、滞在費は研究費から出るという内容の書類を出した）
* 3ヶ月分の銀行口座の明細 (大学からの振り込み履歴が必要)
* 健康保険証明書 (Mawista Studentに加入した)
* (住民登録証) Meldebescheinigung
* (賃貸契約の詳細) Wohnraumnachweis
* biometric photo (申請手続き中に撮影できた)

### 面談で確認される内容

銀行口座の有無を確認された。Wise appの口座を申告した。Wise appは日本で登録してEUR口座を開けばEU圏で使える口座情報（IBAN）などが発行されるのでおすすめ。Revoultなども問題ないと思われる。
最も重要なのは生活費の証明だった。
一度、3/4(水)の10:30に呼び出されて手続きしたが、　経費支給証明書だけでは担当者が納得せず、3/6金曜日 8:30に再度訪問し、銀行明細の印刷したものを提出した。日本語、日本円で記載されたものだが向こうが勝手に翻訳してくれた。
この辺は閉鎖銀行を使うともっと楽かもしれないと思った。理科大の規則では閉鎖銀行に1年分の金額を入れるのは無理かもしれないが。閉鎖銀行を使うと保険もついてくるので、できるならそっちの方が楽そう。とにかくお金の問題の証明が一番やっかいだった。

質問例:
* 毎月いくら支給されるか
* 支給期間
* どのように支給額は計算されるのか

### その他

## Residence Permit発行

全ての書類が受理されるとその場では仮滞在証を発行される。正式な物理カードは後日郵送される。
かかった費用は
| 項目               | 費用   |
| ---------------- | ---- |
| Residence Permit | 100€ |
| 仮滞在許可            | 13€  |
| 証明写真             | 21€  |
| 郵送               | 15€  |

支払い: クレジットカード可。
